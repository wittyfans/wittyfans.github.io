主机名由localhost变成bogon的解决办法

发表于 2017-07-13 | 分类于 技术 | 阅读次数

主机名由localhost变成bogon的解决办法

  • 修改你的 DNS 为公共 DNS，例如 114.114.114.114 或者谷歌的 8.8.8.8（我切换DNS便解决了问题）
  • 编辑主机名字

为什么会出现这个问题

终端在初始化时，需要确定主机的名字，此时终端会向 DNS 服务器反向查询本机 IP 地址，然后把得到的名字作为自己的主机名。如果没有反向解析的结果，就会显示本机设置的主机名。那么 bogon 是什么意思呢？一个 DNS 反向解析名为 bogon 的 IP 地址不会显示在 traceroute 中。我猜题主可能在一个路由器组织的局域网环境中，并且你的 DNS 地址是你路由器的地址，这样终端反向查询你本机 IP 地址得到的结果是 bogon，于是你在终端里显示的主机名就变成了 bogon。

每个人都应该具备的技能：讲故事

发表于 2017-05-29 | 阅读次数

我觉得讲故事的能力是人类拥有的最神奇的能力之一，在我们社交生活中，讲故事几乎是太熟悉不过了。一个好的故事能我们帮我们交流想法、促进沟通，感同深受。

应花更多的时间、注意力去学习讲更好的故事，许多人至今在生活、工作中没有这样的能力，这是因为大部分人都没有花心思在这一点上。

很多年以前人类就是依靠讲故事来交流的，那时候都没有手机、电脑。

以下是一些学习如何讲故事的建议：

  • 讲故事的时候，多利用数据、图标、研究。
  • 头脑中对故事的脉络、架构要清晰，你要懂得故事是如何生效的、哪些地方让故事精彩，什么地方是高潮。
  • 什么时机都可以讲故事，锻炼你讲故事的吸引力。
  • 让你的故事便于理解。
