Project

一个项目会有五个边界，分别是

  1. Cost
  2. Time
  3. Scope, quality
  4. risk（new
  5. resources（new

有的时候，有些边界并不起到作用，因为项目也是遵守自然法则的。如果其中一个发生变化，你就需要重新审视除变化范围外的一项或几项。

Scope

  1. product scope ／ 项目需要交付或者生产的产品
  2. Project scope ／ 项目需要完成的事情
  3. Total scope ／ 包括上面所有

Project manager

  1. technical skills
  2. Project management process skill
  3. Leadership skills

Roles of PM

  1. Task and Human
  2. Integrator and technical expert
  3. Big picture and detail
  4. Politics
  5. Projects vs function

Project management

  1. Planning
  2. Scheduling
  3. Managing
  4. Coordinating
  5. Leading
  6. Communicating
  7. Controlling

Process Groups

  1. Initiate
  2. Plan
  3. Execute
  4. Monitor and Control
  5. Close

Initiating

项目的开始阶段，我们需要找出我们的Stakeholder（that are actively involved in the project），比如我们的客户， 这个阶段你需要考虑好所有参与了，或者会影响这个项目的人，确定他能造成多大的影响，并仔细考虑好相应的对策。我们可以借助一个表格来帮助自己完成这件事情。
下面举例一个项目来说明一下
The project will include:
– installing new turf
– putting in all new lighting
– adding 28% more seats
– constructing a new parking lot
– installing a new score board
