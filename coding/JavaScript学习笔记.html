数组 🔢

  • 通过[x,x,x,x]定义数组，中间使用,分开
  • 通过.length 这个方法取得数组的长度（从0开始计数）
  • 通过[]定义空的数组，空的数组长度为0
  • JS的数组是无类型的
  • 元素可以是任意类型
  • 对象的数组与数组的数组是可以的，数组允许嵌套
  • 数组的索引范围是0-0的32次方-1
  • JS的数组是动态的，可以随时加一减一,这也意味下面的代码是可行的
  • JS数组不要求是连续的，不连续的数组被称为稀疏数组
  • 数组本质上来说就是特殊的对象，但用索引访问数组，比访问对象的属性要快

数组的读写

  • 数组自动维护自己的长度，即length，这是数组与对象明显不同的地方
  • 请清晰的分辨数组的索引与对象的属性名
    • 所有的索引都是属性名
    • 0-0的32次方-2之间的整数属性名才是索引
  • 数组索引仅仅是JS中对象属性名的一种特殊类型，这意味着
    • JS中的数组不会有越界的错误
    • 错误的输入会得到Underfine
  • ES5中的数组可以定义数组的getter和setter方法

稀疏数组

  • 稀疏数组即length的值大于元素的个数
  • delete操作可以生产稀疏数组
  • 稀疏数组的查找速度和一般的数组一样
  • 数组直接量[5,,4]中省略值不会创建稀疏数组 ，它会让省略值为underfine
  • 数组直接量中使用连续的都好会得到稀疏数组

数组长度

  • 即length的值，因此与对象不同，当你直接更改length值的时候，位于这个长度之后的元素值会被删去
  • 当长度更改为大于元素个数的时候，会增加一个空的区域

添加与删除数据

  • Array.Push() 末尾添加元素
  • Array.unshift() 首部插入元素
  • Array.Pop() 删除末尾的元素

遍历数组

  • 遍历的时候，数组的长度建议只查询一次
  • 应跳过Null，undefined元素
  • For-in语句可以用来应对稀疏数组，它会滤掉不存在的索引
  • For-Each语句可以传入函数给每一个值

判断数组

  • 利用Array.isArray()方法
  • isInstanceOfArray()方法，只适用于简单的情形，当在Web浏览器环境中，情况会变得很复杂
  • 还可以检查对象的类属性，例如

类数组对象

  • 可以把一个常规对象增加属性变成类数组对象
  • 类数组对象也可以遍历
  • 客户端JS中，一些DOM方法也返回类数组对象
  • JS的数组方法适用于类数组对象

作为数组的字符串

  • 通用的数组方法可以用在字符串上
  • 字符串是不可变值，即只读
  • push、sort、reverse、splice等方法在字符串中是无效的

数组的常见方法

🔧 Push与Pop方法

  • Push方法操作后返回数组的长度
  • Pop方法操作后返回它删除的值

🔧 unshift与shift方法

  • unshift() ,头部增加元素，所有索引增加，返回长度。且插入顺序和所给顺序一致
  • shift（）删除第一元素，将其返回

🔧 Map方法

map 方法会迭代数组中的每一个元素，并根据回调函数来处理每一个元素，最后返回一个新数组

🔧 reduce方法

方法 reduce 用来迭代一个数组，并且把它累积到一个值中。

  • 两个参数，第一个参数是化简操作的函数，第二个参数是可选的传入的初值
  • 第一次调用的时候，会把第二个参数的值传给第一个参数
  • 第二次调用的时候（遍历到第二个值的时候），会把第一步1的结果加上函数执行后的结果加起来重新复制给第一个参数
  • ReduceRigth（）方法是一样的原理，处理的顺序相反而已

🔧 filter

filter方法用来迭代一个数组，并且按给出的条件过滤出符合的元素
filter 方法传入一个回调函数，这个回调函数会携带一个参数，参数为当前迭代的项（我们叫它 val ）。
回调函数返回 true 的项会保留在数组中，返回 false 的项会被过滤出数组。。

🔧 Sort方法

使用 sort 方法，你可以很容易的按字母顺序或数字顺序对数组中的元素进行排序。
与我们之前用的数组方法仅仅返回一个新数组不同， sort 方法将改变原数组，返回被排序后的数组。

sort 可以把比较函数作为参数传入。比较函数有返回值，当 a 小于 b，返回一个负数；当 a 大于 b ，返回一个正数；相等时返回0。

如果没有传入比较函数，它将把值全部转成字符串，并按照字母顺序进行排序。
